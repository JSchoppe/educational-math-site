Center of Gravity

When a torque is applied to an object it will always rotate about its center of gravity. This is the average location of the object's mass. This center of mass can be calculated by approximating an object shape with primitive shapes where the center of gravity is easy to calculate. Pivoting about the center of gravity ensures physics simulations are accurate to reality.

Example Input:

Circle r=3/2 @(3/2, 3/2)

Rectangle from(3/2, 0) to(9/2, 3)

Triangle a(3/2, 3) b(9/2, 3) c(9/2, 4)

Hole r=1/2 @(3/2, 3/2)

Example Output:

X = (3π + 66) / (2π + 24)

Y = (21π + 296) / (14π + 168)

Bezier Curves

Bezier curves are a way to store curves of any shape with relatively little data. A bezier curve consists of nodes and control vectors that control the shape of the curve from node to node. Bezier curves are a great solution for rendering curves that are self-interesecting, or otherwise not one-to-one functions on any axis.

Example Input:

Node(0, 0) Control(0, 1)

Control(4, 1) Node(4, 4) Control(4, 6)

Control(4, 1) Node(6, 1)

Example Output:

(Graph)

Lagrange Interpolating Polynomial

Given a series of coordinates in 2D space, a polynomial function can be generated that will pass through every coordinate. This can be used to approximate the value at other data point along the interval.

Example Input: (0, 1), (2, 2), (3, 4)

Example Output: ⨍(x) = 0.5*x² - 0.5*x + 1

Spline Generation

Splines are used to connect a series of given points. They are piecewise meaning that each connection between two points is designated its own function. Splines are a quick way to generate descriptive curves while maintaining control over function complexity.

Example Input: (0, 0), (1, 1), (2, 4)

degree=cubic

condition=natural

Example Output:

⨍(x)[0, 1] = 0.5*x - 0.5*x³

⨍(x)[1, 2] = 1 + 2(x-1) + 3/2(x-1)² - 1/2(x-1)³
