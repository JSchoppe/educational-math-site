What is Math Vision?

Math Vision is a static website designed to experiment with animation and visualizations as a tool to teach mathematics in a more general way. The topic matter is currently targeted towards college-level math concepts. The goal is to build strong mental models by directly seeing how changing inputs modifies problems. I hope this project can help students map high level concepts to something beyond just formula memorization.

This tool is currently created and maintained by me, Jethro Schoppenhorst. You can learn more about me on my own website.

Is this project open source?

Yes. To promote evolution of presentation in academia, I have made the project completely open under the MIT license. Feel free to reuse any of the components on this website for any purpose.

What tools were used to make this?

This website runs using Amazon Web Services. There is no backend server code. Everything on this website has been written in plain HTML/CSS/JavaScript. Special thanks to the contributing developers for Blender, Inkscape, Paint.NET, VS Code, and Google Fonts.
